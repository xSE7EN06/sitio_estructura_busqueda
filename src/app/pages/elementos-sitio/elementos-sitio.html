Elementos del Sitio Web

Entiende las piezas fundamentales que componen la estructura de cualquier página web moderna.

La Anatomía Web

Al igual que un edificio necesita cimientos, paredes y techo, un sitio web se construye sobre bloques estructurales definidos. Esta organización permite que los navegadores entiendan el contenido.

Header (Encabezado)

Contiene el logo y navegación. Es la identidad del sitio.

Main (Cuerpo)

Donde vive el contenido único y relevante de la página.

Footer (Pie)

Información legal, enlaces secundarios y cierre de página.
